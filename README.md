## 🔮 游戏效果展示

### 玩法说明

全屏游戏

- 按“A""D"左右移动视角，鼠标滚轮调整俯视程度，鼠标右键点击要去的地方

- 传送门按”E"键开始传送，按“B"调出装备栏与任务栏
- 数字键（非小键盘上的）1~6对应使用快捷栏上的物品
- 按"ESC"返回主菜单



- 内测使用
  - ”S“键进行存档，”L"键加载存档
